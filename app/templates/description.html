Description of the project

This is the final version of the semester project from the Computer Programming 2 subject - CeneoScraper. The core functionality of this app is a web scraper implemented using the bs4 library, used to scrape product reviews from the Polish website ceneo.pl, summarize and analyze this data with charts, which is accomplished using pandas/numpy. In this final version, the functionality is encapsulated in a Flask web application, providing a more user-friendly UI to access the functionality. On top, the app has been hooked with functional user accounts, and logging in is necessary to use the app. Each user has access only to their own data, and user passwords are, of course, stored as hashes only. App data is persistent thanks to the use of an SQLite database, and SQLAlchemy has been used as an ORM to model the relational schema and to interact with the database.

To use the application, first register and log in. Once inside, navigate to the extraction page, and enter a ceneo product code into the input field. After a while, the extraction will be complete and you will be redirected to the product page, where you can see all the reviews. You can use the filters to analyze the reviews yourself, as well as download the data in JSON format and view the generated charts for the product. On the Your Products page, you can find all products you have previously extracted using your account.
